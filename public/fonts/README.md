# Font Installation Required

Please place your `Cocogoose-Pro-Bold-Italic-trial.ttf` font file in this directory.

The application will use a fallback font until the Cocogoose Pro font file is added here.

**Expected file location:**

```
    /public/fonts/Cocogoose-Pro-Bold-Italic-trial.ttf
```

Once you add the font file, the application will automatically use it.
